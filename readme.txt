Show multiple instances widget with the assigned links from Backlinks site for page content and textlinks on different sections, pages and posts.

== Description ==

This plugin is like original **Backlinks** wp plugin, but this have new cool features that can widgetize the links with the key for every page.  

This plugin contain the admin panel like original backlinks plugin, but only used for "My Content Link Pages" and the new cool features that can widgetize the links with the backlinks key for every page.
Now you can use a widget for every page published at Backlinks.com with the key provided in every widget's options.

You can read, follow, donate and comment about this plugin at [wpebacklinks page](http://www.netmdp.com/wpebacklinks) or visit author page for more wordpress things: [NetMdP](http://www.netmdp.com)

== Installation ==

You can either install it automatically from the WordPress admin, or do it manually:

1. Unzip "WPeBacklinks" archive and put the folder into your plugins folder (/wp-content/plugins/).
1. Activate the plugin from the Plugins menu.
1. Use your Widgets settings to configure

== Frequently Asked Questions ==

= Can I use this plug-in without Backlinks plugin? =

YOU MUST. This plugin replace the functions in the original plugin from Backlinks.com, then if you activate both plugins some errors must occurs.
I repeat: you must deactivate Backlinks.com plugin before activate this.

= Can use this plugin, without register at Backlins.com? =

You can use anyway, but is useless without the key provided for Backlinks.com.  You can use this link for register: **[Backlinks.com](http://www.backlinks.com/?aff=52126)**
